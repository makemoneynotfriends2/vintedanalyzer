# Market Scanner - DEPLOYMENT

## WAS IST DAS?
Eine einfache HTML-Seite die auf Vercel läuft. 
Aktuell mit Demo-Daten, später erweiterbar mit echtem Backend.

## DEPLOYMENT (3 SCHRITTE):

### 1. GitHub Repo erstellen
- github.com → New repository
- Name: `market-scanner` 
- Public
- Create

### 2. Files hochladen
- Klick "uploading an existing file"
- Upload NUR `index.html` (diese eine Datei!)
- Commit

### 3. Vercel deployen
- vercel.com → Sign up with GitHub
- New Project
- Wähle `market-scanner` Repo
- Deploy

## FERTIG!
Die App läuft jetzt auf `https://market-scanner-xyz.vercel.app`
